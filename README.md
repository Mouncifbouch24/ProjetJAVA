# Projet JAVA – Analyse d’un fichier CSV

## Description
Ce projet Java permet de lire un fichier CSV et de calculer des statistiques simples sur les colonnes numériques.

Le programme analyse les données du fichier, détecte les colonnes contenant des valeurs numériques, puis calcule pour chaque colonne :
- la moyenne
- la valeur minimale
- la valeur maximale
- le nombre de valeurs utilisées

Les résultats sont affichés dans la console et enregistrés dans un fichier texte nommé `statistiques.txt`.

---

## Objectif du projet
L’objectif principal de ce projet est de mettre en pratique les bases de la programmation orientée objet en Java à travers un cas simple d’analyse de données.

Ce projet permet de travailler sur :
- la lecture de fichiers CSV
- la manipulation de listes
- le calcul statistique
- l’écriture de fichiers texte
- l’organisation du code en plusieurs classes

---

## Fonctionnalités
- Lecture d’un fichier CSV
- Conservation des en-têtes du fichier
- Détection des colonnes numériques
- Calcul de la moyenne, du minimum et du maximum
- Ignorance des cellules vides
- Affichage des résultats dans la console
- Génération d’un rapport dans `statistiques.txt`
